[en] Follow the instructions on this page to override Airflow configuration options for new and existing environments.

[en] When you create or update an environment, you can override Apache Airflow configuration options with different values. By doing so you can adjust the Airflow instance to your needs and requirements.

[en] Override Airflow configuration options for new environments

[en] You can override Airflow configuration options when you create your environment. For more information, see Create environments.

[en] Override Airflow configuration options for existing environments

[en] To override Airflow configuration options for an existing environment:

[en] Console

[en] To override Airflow configuration options for an existing environment:

  1. [en] In Google Cloud console, go to the Environments page.

  2. [en] In the list of environments, click the name of your environment. The Environment details page opens.

  3. [en] Go to the Airflow configuration overrides tab.

  4. [en] Click Edit.

  5. [en] Enter the Section, Key, and Value for the Airflow configuration option that you want to change.
