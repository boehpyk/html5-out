Air Filtering

To connect the Acme BMS to an external air filtering system: no changes here

The Acme 100 is designed to manage the air flow automatically. But if there is a system failure, you can manage it manually by using the CORE-S console installed in the main room. This has wired connections to all of the air flow hardware so that you can take control.
